## Keyboard interactions 
Only the call to action can receive focus if included.

    palette="light",

  <thead>
    <tr>
      <th>Key</th>
      <th>Result</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Tab</td>
      <td>Moves focus to the call to action (if included)</td>

    <tr>

      <td>Selects the call to action (if included)</td>
