# Read time

Displays an estimated reading time for content.

## Usage

```html
<script type="module">
  import '@rhds/elements/rh-readtime/rh-readtime.js';
</script>

<rh-readtime selector="#article">%t min read</rh-readtime>
```

See the [RHDS docs](https://ux.redhat.com/elements/readtime/) for more information.
