Brand/UI icon library

Overview

Icons are small visual representations that help us depict a variety of ideas, technologies, and interactions. Our icons are open and straightforward, just like our brand. This library includes a set of brand icons and a set of UI icons. To see the repo of brand icons, visit this page. To see the repo of UI icons, visit this page.

  • Brand icons - Large icons that represent general and technical concepts
  • UI icons - Small icons used for interactions or states within website or product interface components

Get started

Available brand and UI icons can be accessed in the Libraries panel in Adobe XD. You can search for an icon by name or by expanding any corresponding section.

Adobe XD

This library is maintained in Adobe XD, it needs to be installed if you want access to icons. After XD is installed, your Adobe account needs to be switched to the Adobe for Teams plan. Contact your manager for further assistance if necessary.

Request an Adobe for Teams plan Learn how to use libraries

Usage

Brand icons

Use brand icons to represent general and technical concepts, their recommended size is between 32px and 96px.

UI icons

Use UI icons to represent interactions or states within website or product interface components. UI icons come in various sizes depending on what component they are used in.

Component states

Use Component States in XD to toggle between different colors, interactions, or states.

Warning
Resizing an icon and switching component states will revert the icon to the default size. Select a component state and then resize.

Brand icon sizing

This library is set up so that all brand icons have a height or width of 64px. When using icons in your designs, some of them will appear smaller or larger than others. You might need to visually adjust them so they have similar proportions when used in close proximity to each other.

UI icon sizing

If you need different icon sizes when creating a new component, most UI icons include Small and Large versions.

Best practices

Inconsistent sizing

Do not use brand icons at different sizes in close proximity to each other.

Inconsistent colors

Do not use brand or UI icons with different colors in close proximity to each other.

Mixing icons

Do not mix UI icons with brand icons and vice versa.

Frequently asked questions

When are new icons added to this library?
New brand assets are released every month, so any new brand icons will be added within one day. When a new design system component is created, any new UI icons will also be added within one day.

How do I request a new icon?
To request a new brand icon, contact the Brand team. To request a new UI icon, contact the Design system team.

How do I request an existing icon be added to this library?
If your desired brand or UI icon is not found in this library, check if it exists in the brand icon repo or UI icon repo first. If the icon does exist in those repos but not in this library, contact the Design system team and it will be added within one day.

Why is a brand or UI icon no longer available?
It might have been removed if brand assets or design system components were deprecated. To find out more, contact the Brand team for missing brand icons or the Design system team for missing UI icons.
