Navigation

Introduction

Navigation helps a user orient themselves and enables them to browse through content or complete a task.

Navigation patterns

There are several patterns available, each was designed to serve a specific purpose or solve a user need.

Using navigation

When thinking about how to use navigation, consider how a user would interact with each pattern.

  • Primary and Secondary navigation allow a user to navigate between pages
  • Jump links allow a user to navigate through an entire page of content
  • Tabs and an Accordion allow a user to navigate through a fixed area of content

Primary navigation

Component

Primary navigation contains a slot for a logo, menus or links, and slots for utilities that are placed on a black background.

Usage

Primary navigation allows a user to move between the most important pages of a website. It is positioned at the top of a page and contains links to the most important content. It is persistent in all environments, meaning that it sticks to the top of a window as a user scrolls. The only exception is when Secondary navigation is positioned below it. It is not mandatory to use Primary navigation on every page, but there must be some form of navigation visible to a user at all times, even a logo that lives on its own.
